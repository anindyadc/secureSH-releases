# secureSH — Releases

Binary releases for [secureSH](https://github.com/anindyadc/secureSH), a Tauri-based SSH session manager.

Download the installer for your platform from the [Releases](../../releases) page.

| Platform | File |
|---|---|
| macOS Apple Silicon | `secureSH_*_aarch64.dmg` |
| macOS Intel | `secureSH_*_x64.dmg` |
| Windows 64-bit | `secureSH_*_x64-setup.exe` |
| Windows 64-bit (MSI) | `secureSH_*_x64_en-US.msi` |
| Linux (Debian/Ubuntu) | `secureSH_*_amd64.deb` |
| Linux (AppImage) | `secureSH_*_amd64.AppImage` |

Source code is kept in a private repository.
